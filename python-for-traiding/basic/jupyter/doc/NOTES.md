# Time Value of Money

## Present Discounted Value

```
PV(T0) > PV(T1)
PV(2020) = 1000$
PV(2021) = 1000$ * 0.96 = 960$
  where Inflation=4%
```

## Future Value


```
PV(2020) = 1000$
5% annualy
FV(2021) = 1000$ * 1.05 = 1050$
```

```
FV = P * pow(1 + i, n)
FV - future value
P - principal amount
i - interest rate
n - time in years
```
